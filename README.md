# iWeibo
This is weibo software code.<br/>
功能包括：浏览微博，转发微博，评论微博，查看微博详情，下拉刷新，上拉加载，声音开关，数据离线缓存，账号退出。<br/>
所有图片来源新浪微博，仅用作学习交流，如有侵权烦请告知，将立即撤销。
